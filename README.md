# Raj Thal Vision

A high-end, storytelling-driven restaurant experience crafted for **Hotel Raj Thal**, Surat. This project combines cutting-edge web technologies with traditional Indian culinary heritage to create a cinematic digital journey.

## 🌟 Features

- **Cinematic Hero**: Frame-by-frame scroll animations using GSAP and high-resolution canvas rendering.
- **Interactive 3D Thali**: A deconstructible 3D model of a traditional Indian Thali built with React Three Fiber and GSAP.
- **Narrative Storytelling**: A flowing layout that tells the heritage story of Raj Thal, Surat.
- **Premium UI/UX**: Custom typography, dark-noir aesthetics, and smooth-scrolling experiences.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.

## 🛠️ Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **3D Graphics**: [Three.js](https://threejs.org/) with [React Three Fiber](https://docs.pmnd.rs/react-three-fiber)
- **Animations**: [GSAP](https://gsap.com/) (ScrollTrigger)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see the result.

## 🖼️ Visual Identity

The site uses a palette of **Deep Obsidian (#050505)** and **Regal Gold (#D4AF37)** to evoke a sense of luxury and tradition. The typography features a blend of serif elegance and clean sans-serif modernism.
